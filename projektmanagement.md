# Projektmanagement {#sec-projektmanagement}

::: {.callout-note title="Zusammenfassung"}

Dieses Kapitel wird derzeit im Rahmen des 4. Booksprint erarbeitet.

:::

## Einleitung

Das Management von IT-Projekten umfasst die gezielte Planung, Steuerung und
Überwachung von Vorhaben zur Entwicklung oder Einführung neuer Software oder
Hardware. Im Gegensatz zum allgemeinen IT-Management (@sec-management) handelt es
sich um zeitlich begrenzte Vorhaben mit einem definierten Start- und Enddatum.
