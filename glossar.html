Anhang A — Abkürzungsverzeichnis

APC
Article processing charge, gebräuchlich auch als publication fee
API
Application Programming Interface, Programmierschnittstelle
BMS/LMS
Bibliotheksmanagementsystem, gebräuchlich auch als Library Management System
CMS
Content-Management-System
COUNTER
Counting Online Usage of NeTworked Electronic Resources
DDC
Dewey-Dezimalklassifikation, Klassifikation für die Sacherschließung von Bibliotheksbeständen
DMP
Datenmanagementplan
DNS
Domain Name System. Auflösung von Server-Namen zu IP-Adressen
DOI
Digital Object Identifier, eindeutiger und dauerhafter digitaler Identifikator für Objekte
EDIFACT
Electronic Data Interchange for Administration, Commerce and Transport
ELN
elektronisches Laborbuch
ERM
Electronic Resource Management, Verwaltung von Lizenzinformationen zu E-Ressourcen
ERP
Enterprise-Ressource-Planning, Ressourcen wie z.B. Finanzen unternehmerisch überwachen und planen
FDM
Forschungsdatenmanagement
FID
Fachinformationsdienst
FIS
Forschungsinformationssystem
GOKb
Global Open Knowledgebase, Freie Datenbank zu E-Ressourcen, insbesondere für ERM
ID
Identifikator, eindeutige Referenz auf einen Datensatz
IDM
Identity Management, das Speichern von Metadaten zu Personen
IIIF
International Image Interoperability Framework, Standard für digitale Bilder
KBART
Knowledgebases and related tools, Datenformat zum Transfer von Metadaten
KDSF
Kerndatensatz Forschung, ein Standard für Forschungsinformationen für das deutsche Wissenschaftssystem
LLM
Large Language Model, eine statistisches Sprachmodell zur Erzeugung von Text
LZA
Langzeitarchivierung
MARC
MAchine-Readable Cataloging, ältestes und noch immer wichtigstes bibliothekarisches Austauschformat
NFDI
Nationale Forschungsdateninfrastruktur
OAI-PMH
Open Archives Initiative Protocol for Metadata Harvesting
OCR
Automatische Erkennung von Texten in Bildern (Optical Character Recognition)
OPAC
Online Public Access Catalogue, Katalog einer Bibliothek
PDA
Patron-Driven-Acquisition
PID
Persistent Identifier, eine eindeutige und dauherhafte Benennung einer digitalen Ressource
SIP2
Protokoll zur Anbindung von Selbstbedienungsautomaten an Bibliothekssysteme
SRU
Search/Retrieve via URL, ein technischer Standard für Suchanfragen in Bibliothekskatalogen
SWORD
Simple Webservice Offering Repository Deposit
SaaS
Software as a Service, Software und Hardware bei externem Dienstleister
WCAG
Web Content Accessibilty Guidelines
